KeyPic


KeyPic was a project I worked on with my friend Ben Rowley at HackMIT 2017. At the time, we were both really interested in physical security systems. Specifically, we were both interested in locks and keys. With this in mind, we came up with the idea of creating an application where you could generate an STL file for a key from a single picture. Once the STL file was generated, you could easily 3D print it yourself or send it out to be printed by a professional 3D printing service.
